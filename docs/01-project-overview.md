# Project Overview

Kourai is an active early-stage product prototype for AI-assisted social content workflows. It explores how marketing teams could move from goals to strategy, drafts, review, reporting, and performance-informed recommendations.

PHINN is the first specialized agent inside Kourai. It began as an experimental social-media persona and now serves as the first validation path for agent behavior, workflow orchestration, safety review, account-aware memory, and learned voice.

X/Twitter is the current platform focus. Instagram, TikTok, and additional platforms are planned for later phases after the backend learning loop, approval controls, and product interface are stronger.

## Current Stage

- Core workflow orchestration and SQLite-backed memory are implemented.
- Historical-content classification, learned voice, effective voice, and chat orchestration are active prototype capabilities.
- A local chat-based interface can communicate with the orchestration layer for development and testing.
- Additional platform integrations, authentication, and production deployment remain planned work.

## Product Direction

Kourai is intended to act as a middle layer between social platforms, marketing teams, analytics, and advertising workflows. The system is designed to preserve human approval while using AI to help retrieve context, draft content, review risk, and learn from prior performance.
