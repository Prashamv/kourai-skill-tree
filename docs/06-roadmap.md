# Roadmap

## Implemented and Validated Locally

- PHINN persona and X/Twitter workflow foundation
- Modular Python task orchestration
- Strategy, post, reply, enhancement, and gatekeeper workflows
- SQLite-backed records for workflow memory
- Account-linked database planning
- Migrations, backups, and fresh-database validation
- Historical post ranking foundation
- Human-readable database exports
- Historical content classification support
- Learned and effective voice backend
- Database-aware chat orchestration
- Persistent chat sessions, messages, and action records
- Approval-required action routing and live-posting protection

## Partially Implemented or Being Expanded

- Larger historical datasets for voice learning
- Stronger original, reply, and quote/commentary sample coverage
- Confidence calibration for learned voice and classification
- Deeper engagement-to-style correlations
- Effective voice explanations for users
- Broader natural-language coverage and multi-turn reference handling
- More granular approval policies and action previews

## Planned

- Richer platform-specific voice profiles
- Expanded platform actions beyond the first X/Twitter focus
- Role-based permissions
- Production authentication and deployment
- Multi-platform campaign coordination
- Controlled scheduling or publishing only after explicit authorization
